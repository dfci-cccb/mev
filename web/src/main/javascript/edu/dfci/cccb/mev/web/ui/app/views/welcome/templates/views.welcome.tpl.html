Welcome to the Web MEV

Web MeV (Multiple Experiment Viewer) is a web tool for genomic data analysis. Web MeV is being built to meet the challenge of exploring large public genomic data set with intuitive graphical interface for analysis. As the MeV team continues to add functionalities into the application, we request the users to provide us with feedback on the look and feel of the Web MeV infrastructure.

Data Sets

Web MeV incorporates TCGA gene expression data for direct import into analysis. We currently house level 3 gene expression array data for all 16 cancer types. The user can create sample sets from clinical information for focused analysis. We currently use google OpenRefine technology to allow the users to sift through TCGA. A tutorial for openrefine can be found here. Users can also directly access GEO data sets through search interface.

The user can also upload their own data of interest for analysis.

Currently, log in is not required to use Web MeV

How to use

MeV is still open-source. You can download the entire stable version and run locally on your own computer, or right on your browser.
